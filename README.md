# StreaKing - Android Habit Tracker

A streamlined Android application designed to track daily habits through a streak-based system. This project demonstrates a clean separation of concerns between business logic and UI components, utilizing modern Android development standards.

## 🛠 Engineering Approach
This project is built with a focus on **scalability** and **maintainability**:
- **Logic Isolation:** The streak calculation is decoupled from the UI via the `StreakManager` object.
- **Modern UI Stack:** Developed using **Jetpack Compose** for a reactive and declarative user interface.
- **State Management:** Utilizes Kotlin's `remember` and `mutableStateOf` for efficient UI state handling.

## 🚀 Features
- **Real-time Counter:** Visual feedback of current streaks with an intuitive interface.
- **One-Tap Update:** Simplified user interaction to encourage habit consistency.
- **English Localization:** Designed for international accessibility and professional standards.

## 💻 Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Architecture:** Simplified MVC (Model-View-Controller) approach for rapid prototyping.
